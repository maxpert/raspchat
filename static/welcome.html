Welcome to Raspberry Pi Model B+ chat server

Here is technology stack:

  • Raspberry Pi 1
  • Raspbian Jessie Lite 4.1
  • Backend compiled on Golang 1.5
  • Websockets!
  • WebRTC for file sharing(coming soon)
  • MarkdownIt for all markdown things (see https://daringfireball.net/projects/markdown/)
